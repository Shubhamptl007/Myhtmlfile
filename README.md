A clean, fast, and feature-rich task management web application built with pure Vanilla JavaScript.

Features

Full CRUD operations
- Task categories
- Priority levels
- High-priority notifications
- Mark tasks as completed
- Real-time search
- Sorting
- Dark / Light theme toggle
- Responsive design
- Local persistence
- Clean object-oriented
- Input validation
- Smooth animations, modern UI, keyboard shortcuts

Tech Stack

- HTML5
- CSS (Tailwind CSS via CDN)
- Vanilla JavaScript (ES6+)
- Font Awesome (icons)
- localStorage (persistence)
